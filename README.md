# Gomoku Next

加入道具机制的五子棋游戏（由 TRAE SOLO 完成）

[🎮 在线体验](https://gomoku-next.netlify.app)

## 项目概述

Gomoku Next 是一个基于 TypeScript 和 Canvas 开发的现代化五子棋游戏，支持「玩家 VS 玩家」和「玩家 VS 电脑」两种模式，
内置丰富的道具系统和 AI 对战功能。

## 核心功能

### 游戏特性

- ✅ 15×15 和 19×19 两种棋盘大小
- ✅ 丰富的道具系统
- ✅ AI 智能对战
- ✅ 游戏状态实时更新
- ✅ 黑暗 / 浅色主题自适应

### 游戏模式

- **玩家 VS 玩家**: 支持两名玩家在同一设备上对战
- **玩家 VS 电脑**: 与内置的 AI 选手对战

### 道具系统

- 强化类
  - 混沌打击：随机移除对手的1颗棋子
  - 精准打击：允许玩家指定移除对手的1颗棋子
  - 加速落子：使玩家在本回合获得额外1次落子机会
  - 策略指引：额外获得1次落子机会，并且系统将推荐3个最优落子位置
  - 钢化我心：使玩家免疫下一次对手发起的移除类攻击
  - 手滑惩罚：限制对手下一回合只能从系统随机给出的3个位置中选择1个落子
- 弱化类
  - 自我失误：随机移除玩家自己的1颗棋子
  - 对手加速：使对手在本回合获得额外1次落子机会

## 使用指南

### 游戏规则

1. 黑白双方轮流落子，黑方先行
2. 先在棋盘上形成横向、纵向或斜向的连续五个棋子者获胜
3. 棋盘满子仍未分出胜负则为平局
4. 道具可通过开启盲盒获得，使用后会产生相应效果

### 操作方法

- **鼠标操作**: 点击棋盘空白位置落子
- **触摸操作**: 触摸棋盘空白位置落子
- **道具使用**: 自动触发或根据提示操作
